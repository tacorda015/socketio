To use Socket.IO

** In Server **
1. npm init
2. npm i nodemon
3. Modify the package.json script to "start": "nodemon server.js"
4. Create a server.js file
5. npm run start
6. npm i socket.io
7. npm install axios
8. modify the server.js file

** In Client **
1. npm install socket.io-client
2. modify the file need real time data
